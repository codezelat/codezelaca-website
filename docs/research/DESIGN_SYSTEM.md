# CCA Homepage Design System

- Background lock: true white outside the hero; hero is `#fdf2f8`.
- Primary gradient: `#cb00f8` to `#710bc0`; primary button surface `#c026d3`.
- Text: headings `#000000`; supporting copy `#4a5565`; footer `#101828`.
- Typography: Poppins for brand, headings, navigation, and controls; Inter for feature rows and footer utilities.
- Desktop heading: 42px / 45px / 600. Mobile heading: 32px / 32-38.4px / 600. Hero mobile: 36px / 39.6px.
- Section label: 18px / 18px / 700, `#710bc0`, 1px neutral border, rounded pill.
- Main container: 1260px max. Desktop outer clone canvas: 1440px. Mobile gutter: 20px.
- Card radius: 20px. Buttons: 10px. Header: 80px. Footer/CTA surfaces: 20px.
- About cards: 1px black border with a second inset outline effect; image radius 18px.
- Feature cards: two 1px `rgba(74,85,101,.31)` outlines separated by 3px.
- Program cards: 1px `rgba(0,0,0,.15)`, 20px radius, 10px padding, 550px desktop height.
- CTA shadow: `0 0 10px rgba(0,0,0,.5)`; header shadow: `0 0 10px rgba(0,0,0,.09)`.

Allowed above-the-fold copy is exactly the source header labels, cohort sentence, hero heading/body, two CTA labels, proof row, and four metric labels. No new hero badge, claim, metric, or CTA is permitted.
