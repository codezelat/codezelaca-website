# About Us Page Specification

## Overview

- Target: `app/about-us/page.tsx` and `components/pages/AboutPage.tsx`
- Interaction model: static editorial content plus recognition carousel

## Structure

- Shared inner hero with `About Us`, the live lowercase promise heading, exact copy, and contact/pathway actions.
- Two alternating editorial image/copy rows: `Who Are We?` and `What is CCA?`.
- Existing six-card differences pattern with heading `What Makes Us Different` and the approved `over 18` wording.
- Promise/Vision/Mission block followed by four proof metrics.
- Shared recognition, consultation and footer.

## Responsive behavior

- Desktop editorial rows use balanced two-column layouts.
- Mobile stacks copy and imagery without cropping text or causing overflow.
- Shared proof sections preserve homepage component geometry and behavior.
